**Project Description**

SQL Editor is a fully-featured, desktop SQL IDE built with Python, Tkinter, and ttkbootstrap.
It delivers a modern, lightweight alternative to heavy database clients by combining an intuitive UI with powerful tooling for writing, debugging, visualizing, and exploring SQL data.

The application connects to MySQL databases and offers multi-tab editing, auto-complete, result pagination, AI-assisted SQL generation, interactive schema diagrams, data visualizations, and system-level database tools â€” all inside a polished interface.

Designed for developers, students, analysts, and DBAs, SQL Editor functions as a complete environment for querying, analyzing, and understanding databases without ever leaving a single window.

**Key Features**

1. **Modern Multi-Tab SQL Editor**
Syntax highlighting, autocompletion, dark/light modes, animated cues, and automatic draft saving.

2. **MySQL Connection Manager**
Save encrypted credentials, switch between connections, and instantly refresh schema information.

3. **Intelligent Query Execution Engine**
Background execution threads, query cancellation, real-time statuses, execution previews, and full query history.

4. **Powerful Result Viewer**
Pagination, sorting, auto-fitting columns, multi-format copy actions, and export to CSV/Excel/Markdown/SQL INSERT.

5. **Schema Browser + Interactive EER Diagram**
Explore databases visually through a draggable, zoomable EER diagram with foreign-key mapping and PNG/PDF export support.

**AI-Powered SQL Assistance (Gemini)**

1. Natural-language-to-SQL conversion (SELECT-only).

2. AI-generated chart queries.

3. Automatic cleaning of AI output.

**Built-In Data Visualization Tools**
Bar, pie, and line charts using matplotlib, with optional hover annotations via mplcursors.

**Workspace Persistence**
Auto-restore tabs, query text, and session state across restarts.

**Server Administration Tools**
User browser, variables, process list, and a mini SQL console.

Polished UX Enhancements
Toast notifications, loading overlays, theme-aware widgets, smart resizing, filters, and smooth layout management.
